Manual de Estatística

Guia completo dos conceitos estatísticos utilizados nesta ferramenta. Use este manual para entender cada teste, métrica e interpretação.

📌 Conceitos Básicos

Média (x̄)

Definição: Soma de todos os valores dividida pela quantidade de valores.

Fórmula: x̄ = Σx / n

Uso: Representa o valor central dos dados. Sensível a valores extremos.

Mediana

Definição: Valor que divide os dados ao meio (50º percentil).

Uso: Mais robusta que a média com dados assimétricos ou com outliers.

Interpretação: 50% dos dados estão abaixo, 50% acima.

Desvio Padrão (DP)

Definição: Medida de dispersão que indica quanto os dados variam em torno da média.

Fórmula: DP = √(Σ(x - x̄)² / (n-1))

Interpretação: DP pequeno = dados concentrados; DP grande = dados espalhados.

Regra 68-95-99.7: 68% dos dados em μ±1DP, 95% em μ±2DP, 99.7% em μ±3DP.

Coeficiente de Variação (CV)

Definição: Razão entre desvio padrão e média, em percentual.

Fórmula: CV = (DP / x̄) × 100%

Uso: Compara variabilidade entre grupos com escalas diferentes. CV < 15% = baixa variação.

🔬 ANOVA (Análise de Variância)

Objetivo: Testar se há diferenças significativas entre as médias de 3 ou mais grupos.

Pressupostos

  • Normalidade: Dados em cada grupo devem ser aproximadamente normais
  • Homogeneidade: Variâncias dos grupos devem ser similares
  • Independência: Observações devem ser independentes

Estatística F

Fórmula: F = MSentre / MSdentro

Interpretação: F > 1 sugere diferenças; quanto maior o F, mais evidência de diferenças.

p-valor < 0.05: Rejeita-se H₀; há diferença significativa entre grupos.

Eta-Quadrado (η²)

Definição: Proporção da variância total explicada pelas diferenças entre grupos.

Escala de efeito:

  • η² < 0.01: Efeito mínimo
  • η² 0.01-0.06: Efeito pequeno
  • η² 0.06-0.14: Efeito médio
  • η² > 0.14: Efeito grande

Teste de Tukey HSD (Post-hoc)

Objetivo: Após ANOVA significativa, identifica quais grupos diferem entre si.

HSD: Diferença Honestamente Significativa. Diferenças maiores que HSD são significativas.

Controla: Taxa de erro familywise (evita falsos positivos com múltiplas comparações).

🎯 Intervalo de Confiança 95%

Objetivo: Estimar o intervalo provável contendo a média populacional com 95% de confiança.

Fórmula com σ Conhecido

IC = x̄ ± z* × (σ / √n)

Uso: Quando o desvio padrão populacional é conhecido (raro).

Fórmula com σ Desconhecido

IC = x̄ ± t* × (s / √n)

Uso: Quando usa-se a amostra para estimar (mais comum).

gl (graus de liberdade): n - 1. Maior amostras = IC mais estreito.

Interpretação

Correto: "Há 95% de confiança de que a média populacional está neste intervalo."

NÃO significa: "95% de chance da média estar neste intervalo" (a média é fixa).

📈 Regressão Linear

Objetivo: Modelar a relação linear entre uma variável independente (X) e dependente (Y).

Modelo Linear

Fórmula: Ŷ = β₀ + β₁X + ε

  • β₀ (intercepto): Valor de Y quando X = 0
  • β₁ (inclinação): Mudança em Y para cada unidade de X
  • ε (erro): Variação não explicada pelo modelo

R² (Coeficiente de Determinação)

Definição: Proporção da variância de Y explicada por X.

Intervalo: 0 a 1 (ou 0% a 100%)

  • R² = 0.9: 90% da variância explicada (excelente)
  • R² = 0.5: 50% da variância explicada (moderado)
  • R² = 0.1: 10% da variância explicada (fraco)

Correlação de Pearson (r)

Definição: Mede a força e direção da relação linear entre X e Y.

Interpretação: r = √R² (com sinal)

  • |r| > 0.7: Correlação forte
  • |r| 0.3-0.7: Correlação moderada
  • |r| < 0.3: Correlação fraca

p-valor dos Coeficientes

Teste t: Verifica se β₀ e β₁ são significativamente diferentes de zero.

p < 0.05: Coeficiente é estatisticamente significativo.

Intervalo de Previsão (IP)

Definição: Intervalo para um valor individual de Y (mais amplo que IC).

Uso: Menos preciso que IC porque inclui erro de previsão.

⚠️ P-Valor e Significância Estatística

O que é p-valor?

Definição: Probabilidade de observar dados tão extremos quanto os observados, assumindo que a hipótese nula (H₀) é verdadeira.

Intervalo: 0 a 1 (ou 0% a 100%)

Níveis de Significância

  • p < 0.001: Altamente significativo (***)
  • p < 0.01: Muito significativo (**)
  • p < 0.05: Significativo (*) ← Padrão
  • p < 0.10: Marginalmente significativo (•)
  • p ≥ 0.05: Não significativo (ns)

Interpretação Correta

Correto: "Se H₀ fosse verdadeira, haveria 5% de chance de ver estes dados."

❌ NÃO É: "Há 5% de chance de H₀ ser verdadeira"

💡 Dicas e Boas Práticas

Antes de Analisar

  • Visualize os dados: use gráficos para entender a distribuição
  • Procure outliers: valores extremos podem distorcer resultados
  • Verifique normalidade: testes como Shapiro-Wilk (não disponível aqui)
  • Verifique homogeneidade: teste de Levene (não disponível aqui)

Tamanho da Amostra

  • n < 30: Use t-Student (mais conservador)
  • n ≥ 30: Distribuição normal pode ser assumida (Teorema do Limite Central)
  • Maior n = maior poder estatístico (mais fácil detectar diferenças reais)

Regressão

  • Escopo: não extraia previsões muito além do intervalo de X observado
  • Resíduos: devem ser aproximadamente normais e sem padrão
  • Outliers: verificar pontos com resíduos padronizados > 2

Múltiplas Comparações

Problema: Quanto mais testes, maior chance de falso positivo (erro Tipo I).

Solução: Use Tukey HSD ou outros testes corrigidos (aplicado automaticamente aqui).
