# Lab Científico PIBIC PRO + Statistical Suite

Aplicação web desenvolvida para apoio à análise estatística em projetos científicos, especialmente voltada para estudantes de iniciação científica (PIBIC) e pesquisadores.

A plataforma permite realizar análises estatísticas diretamente no navegador, sem necessidade de software externo.

## Funcionalidades

- Cálculo de média, mediana e desvio padrão
- Intervalo de confiança (IC 95%)
- ANOVA
- Testes pós-hoc
- Regressão
- Visualização gráfica automática
- Exportação de dados em CSV
- Exportação de relatórios em PDF

## Aplicação

O sistema foi desenvolvido para facilitar análises estatísticas em projetos acadêmicos e atividades de iniciação científica.

Pode ser utilizado em áreas como:

- Ciências da saúde
- Educação
- Ciências biológicas
- Projetos PIBIC
- Análise de dados experimentais

## Tecnologias utilizadas

- HTML
- JavaScript
- Service Worker (PWA)
- GitHub Pages
